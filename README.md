# Библиотека научных статей

React приложение для отображения научных статей и публикаций.

##  Функциональность

- Просмотр списка статей
- Детальная информация о каждой статье
- Фильтрация по типу доступа (бесплатный/по подписке)
- Категории и теги статей

##  Технологии

- React
- CSS3
- JavaScript ES6+

##  Компоненты

1. **ArticleList** - список всех статей
2. **ArticleItem** - карточка отдельной статьи  
3. **ArticleDetail** - страница с детальной информацией

##  Дизайн

- Светло-коричневая цветовая гамма
- Темно-коричневый текст
- Закругленные углы элементов

##  Запуск проекта

```bash
# Установка зависимостей

# Запуск в development режиме
npm start

# Сборка для production
